## How it works

The goal of this design is to have the user roll a d20 dice by clicking an external button. The seven-segment display will show the numbers rolling until the user presses the button again, ending the roll. There is also an external reset button for the user to clear the display and roll again. I built this program using ChatGPT, since I am not familiar with Behavioral Verilog, having only taken CSE 100 and learned structural Verilog. My goal was to familiarize myself with how a project using this form of Verilog would be structured to be prepared for the next Verilog class CSE 125. The design is currently incomplete but will hopefully be fully functional in time to submit this for a TinyTapeout slot.
