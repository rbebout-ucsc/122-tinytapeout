# Tiny Tapeout Verilog Project

This project uses a simple 5 bit LFSR to roll a random number 1-20 and show it on a seven-segment display. It uses an external button to roll the value, with a half second delay between rolls to prevent glitching on the display. There is also a reset button to clear the display and reset the registers.
